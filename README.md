# REU_2026 — Statistics Workshop for AWARE-AI / AI-PROWIL

🔗 **Live Site:** [https://dharaden.github.io/REU_2026/](https://dharaden.github.io/REU_2026/)

## About

This repository contains the materials for the 2026 statistics workshop developed for REU (Research Experience for Undergraduates) participants in the AWARE-AI / AI-PROWIL IRES program. The workshop introduces R and Quarto-based workflows for data analysis, visualization, reproducible reporting, and structural equation modeling.

**Instructor:** Dustin Haraden, PhD — Assistant Professor of Clinical Psychology, Rochester Institute of Technology
📧 [dxhgsh@rit.edu](mailto:dxhgsh@rit.edu) | 🦋 [@dustinharaden.bsky.social](https://bsky.app/profile/dustinharaden.bsky.social)

## Workshop Topics

- Getting started with R & RStudio
- Data wrangling with `dplyr` and visualization with `ggplot2`
- Reproducible tables (correlations & regressions)
- Integrating statistics into text with Quarto
- Structural Equation Modeling (EFA, CFA, Path Analysis, LGC, RI-CLPM)

## Getting Started (For Contributors/Developers)

1. Clone the repository: `git clone https://github.com/dharaden/REU_2026.git`
2. Restore the R environment: open `REU_2026.Rproj` in RStudio, then run `renv::restore()`
3. Preview the site locally: `quarto preview`
4. Render for deployment: `quarto render`

> **Requirements:** R (≥ 4.3), RStudio, Quarto CLI
